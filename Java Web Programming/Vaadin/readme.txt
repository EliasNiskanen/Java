Aluksi päästään kirjautumissivulle, siellä toimii joko user tai admin käyttäjätunnukset, salasanat ovat samat kuin käyttäjätunnukset. 

Sitten tullaan mittaustulos näkymään ja vasemmmasta reunasta löytyy klikatun rivin tiedot joita voi muokata, tallentaa keskeyttää muokkauksen tai poistaa. 

Yläpalkista löytyy kolme välilehteä mittaukset, johon sovellus aukeaa, potilaat, jossa toimii muokkaukset ja deletoinnit samalla tavoin kuin mittaustuloksissakin, sekä haku jossa voidaan hakea tietoja etu tai sukunimellä, 
puhelinnumerolla, tässä jäi joku mättäämään vielä mittauksien filtteröinnissä mutta ainakin potilaat saa näkyviin hyvin!
